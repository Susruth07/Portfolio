# Susruth Ganji Portfolio

A professional localhost portfolio website for organizing work experience, achievements, projects, and resume-ready highlights.

## Run locally

From this folder, start the included PowerShell web server:

```powershell
powershell -ExecutionPolicy Bypass -File .\server.ps1 -Port 8000
```

Then open:

```text
http://localhost:8000
```

## Customize

Most resume content is in `script.js`. Replace the sample roles, achievements, projects, skills, and email address with verified details from your resume or LinkedIn profile.
